Question-1: What are the different ways to select an element in the DOM?

You can select an element in the DOM using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getAttribute(), and childNodes().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Returns or sets the HTML content, including tags, inside an element. innerText: Returns or sets the visible text content of an element, excluding hidden text and HTML tags. textContent: Returns or sets the text content of an element, including hidden text, but without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where instead of attaching event listeners to individual elements, you attach a single event listener to a parent element and use event propagation (bubbling) to handle events for its child elements. This improves performance and simplifies code, especially when dealing with dynamic elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event triggered on an element first runs its handler on that element, then propagates upward through its ancestors (parent, grandparent, etc.) in the DOM hierarchy, triggering event handlers on each of them, unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create an element with document.createElement(), add it with appendChild() or insertBefore(), and remove it with remove(). or createElement().
